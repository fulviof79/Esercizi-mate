\question\exonly{
Pendant l'année, un étudiant a obtenu les notes $72$, $80$, $65$, $78$ et $60$. Si l'examen final compte pour un tiers dans la note annuelle, quelle note l'étudiant doit--il obtenir pour avoir une moyenne annuelle de $76$?

}
\solonly{Il devra obtenir la note de $86$}
